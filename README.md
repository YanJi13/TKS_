# Project TKS by Team TKS

TKS means "Tulo Ka Section," which means three sections in Cebuano. The name TKS comes from the fact that all three members of the group are from three different class sections for the term. 

Project TKS is a simple and straightforward prototype application for a small spa. It is meant to be a one-to-one computerized equivalent of the spa's existing logbook, along with an added simple inventory tracking system.

# User Interface
The UI of the application was made with simplicity, ease of use, and a smooth transfer of operations for the users in mind. The logbook's features are present in the app so that the users will be familiar with them,
ensuring a smooth transfer of operations. The app consists of 4 sections: attendance, appointments, inventory, and miscellaneous, each having different CRUD functionalities.
